Definition: HTML, CSS, FTP, and SSH

HTML
HyperText Markup Language. It is a non technically programming language that is used to structure a web page and its content.
CSS
Cascading Style Sheets. It is a code that styles web content.
FTP
File transfer protocol is a way to download, upload, and transfer files from one location to another on the internet and between computer systems.
SSH
Secure Shell Access that enables you to connect with your server and make changes.
